The Spotify Song Recommendation System takes in 1) a user-selected playlist URL and 2) a number of desired song recommendations as input, and generates a list of recommendations based on various song data and audio features.

To learn more about about the main components and implementation of this project, feel free to check out our GitHub repository: GitHub Repository with Data, Feature Acquisition/Engineering Pipeline, and Website Code. If you would like to generate recommendations from a larger database of 1M songs on a localhost server, instructions to do so and the corresponding data can also be found on the GitHub.


This project was built by Angela Kan and Bruno Romani.
